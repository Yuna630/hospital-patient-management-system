# Hospital Patient Management System

A C++ application that simulates patient management in a hospital environment.

## Features

- Emergency Room (FIFO Queue)
- Intensive Care Unit (LIFO Stack)
- Patient severity tracking
- Custom linked list implementation
- Object-Oriented Design

## Technologies

- C++
- Templates
- Inheritance
- Linked Lists
- Stack and Queue Data Structures

## Project Structure

Patient.h / Patient.cpp
SeverePatient.h
Node.h
Stack.h
Queue.h
ICU.h
ER.h
main.cpp

## Concepts Demonstrated

- Object-Oriented Programming
- Data Structures
- Templates
- Dynamic Memory Allocation
- Inheritance and Polymorphism
